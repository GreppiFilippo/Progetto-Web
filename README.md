# Progetto-Web

This branch contains the static version of the "Mensa Campus" website: HTML/CSS pages with no backend.

**Purpose**
- Provide the front-end pages for demo and design (login, registration, menu, admin/user panels).

**Main files**
- `www/index.html` — Home page
- `www/login.html` — Login page
- `www/register.html` — Registration page
- `www/menu.html` — Menu page
- `www/admin-dashboard.html` — Admin dashboard
- `www/admin-menu.html` — Admin menu management
- `www/admin-add-dish.html` — Add dish (static mockup)
- `www/admin-bookings.html` — Admin bookings
- `www/user-dashboard.html` — User dashboard
- `www/user-profile.html` — User profile
- `www/user-bookings.html` — User bookings
- `www/css/style.css` — Main stylesheet
- `www/upload/` — Uploads/resources folder
